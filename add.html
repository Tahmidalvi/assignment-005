What is the Document Object Model (DOM)?

The DOM is a programming interface for interacting with HTML, XML, and other structured documents. It represents the document as a tree structure, allowing developers to access, modify, and create elements and attributes within the document.

How do you select an element from the Dom?

Use document.getElementById('id'), document.querySelector('selector'), document.getElementsByClassName('className'), or document.getElementsByTagName('tagName') to select DOM elements.

What is event delegation in the context of the Dom, and why is it useful?

Event delegation is a technique where you attach an event listener to a parent element to handle events for its children. It's more efficient and easier to manage than adding individual event listeners to each child.

How do you manipulate an element’s attributes and styles using the DOM?

To manipulate attributes in the DOM, use element.getAttribute('name'), element.setAttribute('name', 'value'), and element.removeAttribute('name'). For styles, use element.style.property = 'value' to set and element.style.removeProperty('property') to remove.
